# Test Data Review

## 목적

운영데이터가 개발, 시험, 모델 평가, 샘플 생성, 파인튜닝에 무단 사용되는 것을 방지한다.

## 원칙

- 합성 데이터 또는 가공 데이터를 우선 사용한다.
- 운영 DB 원본 사용은 금지 또는 별도 승인 대상으로 본다.
- 불가피한 경우 책임자 승인, 접근통제, 유출 모니터링, 시험 후 삭제가 필요하다.

## 출력 형식

```md
- 판정:
- 시험 목적:
- 데이터 출처:
- 개인정보 포함 여부:
- 가공/마스킹 방식:
- 사용 기간:
- 삭제 기한:
- 필요한 승인:
```

# Planning Privacy Review

## 목적

기획 단계에서 개인정보 처리, 외부 LLM 전송, 위탁, 국외이전, 보관기간, 파기 기준을 누락하지 않도록 한다.

## 필수 질문

- 개인정보를 수집하거나 이용하는가?
- 수집 항목은 목적상 최소한인가?
- 외부 LLM 또는 외부 API로 전송되는가?
- 개인정보 처리방침 변경이 필요한가?
- 보유기간과 파기 기준이 있는가?
- 관리자 화면 마스킹이 필요한가?

## 출력 형식

```md
- 개인정보 처리 여부:
- 데이터 항목:
- LLM 사용 위치:
- 외부 제공/위탁/국외이전 여부:
- 보유기간:
- 파기 기준:
- 필요한 승인:
```

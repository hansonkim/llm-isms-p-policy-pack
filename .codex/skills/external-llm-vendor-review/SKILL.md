# External LLM Vendor Review

## 목적

외부 LLM, SaaS AI, 클라우드 AI 사용 시 위탁, 국외이전, 보관기간, 모델 학습 사용 여부를 점검한다.

## 확인 항목

- 서비스명
- 입력 데이터 종류
- 개인정보 포함 여부
- 프롬프트/응답 저장 여부
- 모델 학습 사용 여부
- 데이터 보관기간
- 리전/처리 위치
- 위탁/재위탁
- 국외이전
- 계약서/SLA/DPA
- 삭제 요청 가능 여부

## 2026 ISMS-P 개편 대비 추가 점검

다음 중 하나에 해당하면 강화인증 또는 상시점검 대상 가능성을 검토한다.

- 대규모 개인정보처리자
- 이동통신사업자
- 본인확인기관
- 주요 공공시스템운영기관
- 국민생활 파급력이 큰 서비스
- 개인정보처리시스템이 외부 인터넷과 연결된 디지털 자산을 포함
- AI, LLM, 클라우드, RAG, 벡터 DB, 프롬프트 로그 저장 구조 사용

판정:

- 기준 확정 전이라도 개인정보 원문 외부 전송, LLM API 위탁, 국외이전, 프롬프트 로그 저장은 고위험 처리한다.
- 위탁/재위탁, 처리 리전, 보관기간, 모델 학습 사용 여부가 불명확하면 보류로 판정한다.

# External LLM Vendor Review

## 확인 항목

- 서비스명
- 입력 데이터 종류
- 개인정보 포함 여부
- 프롬프트/응답 저장 여부
- 모델 학습 사용 여부
- 데이터 보관기간
- 리전/처리 위치
- 위탁/재위탁
- 국외이전
- 계약서/SLA/DPA
- 삭제 요청 가능 여부

## 출력 형식

```md
- 서비스명:
- 입력 데이터:
- 저장/학습 여부:

- 위탁/국외이전:
```

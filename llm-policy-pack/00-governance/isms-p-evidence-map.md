## 인증 유형 및 제도 변경 증적

| 변경 항목 | 확인 기준 | 남겨야 할 증적 |
|---|---|---|
| 일반 ISMS-P 적용 | 2023.11 인증기준 안내서, 고시 별표 7 | 적용 기준표, 통제 매핑표, 심사 범위 정의 |
| 인증의 특례 또는 간편인증 적용 | 2024.7.24 고시, 별표 7의2, 별표 7의3, KISA 간편인증 세부점검항목 | 적용 대상 판정 기록, 간편인증 세부점검항목 매핑, 제외/완화 사유 |
| 2026 개편 대비 | ISMS-P 인증제 실효성 강화방안, 후속 고시 모니터링 | 강화/표준/간편 예상 분류, 상시점검 대비 계획, 기술심사 대비 증적 |
| AI·LLM·클라우드 특화 심사 가능성 | AI, LLM, 클라우드, RAG, 벡터 DB, 프롬프트 로그 저장 여부 | 아키텍처 다이어그램, 데이터 흐름도, 위탁/국외이전 검토서, 로그 보관·파기 기준 |

## 감사 대응 메모

LLM 관련 통제는 별도 시스템으로 보더라도 다음 자산을 식별해야 한다.

- LLM 서비스
- LLM API Key
- 프롬프트 로그 저장소
- 응답 로그 저장소
- RAG 문서 저장소
- 벡터 DB
- 외부 LLM 벤더
- LLM을 호출하는 애플리케이션
- LLM 결과를 저장하는 DB 또는 파일 저장소

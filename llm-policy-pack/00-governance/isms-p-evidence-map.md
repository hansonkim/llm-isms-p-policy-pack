## 감사 대응 메모

LLM 관련 통제는 별도 시스템으로 보더라도 다음 자산을 식별해야 한다.

- LLM 서비스
- LLM API Key
- 프롬프트 로그 저장소
- 응답 로그 저장소
- RAG 문서 저장소
- 벡터 DB
- 외부 LLM 벤더
- LLM을 호출하는 애플리케이션
- LLM 결과를 저장하는 DB 또는 파일 저장소

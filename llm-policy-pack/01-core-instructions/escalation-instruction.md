# 보류·금지·사고 후보 에스컬레이션 지침

## 보류 대상

- 데이터 등급 불명확
- 목적 불명확
- 보관기간 불명확
- 파기 기준 없음
- 외부 LLM 보관/학습 정책 불명확
- 국외이전 여부 불명확
- 위탁/재위탁 여부 불명확

## 금지 대상

- 승인 없는 외부 LLM에 개인정보 원문 입력
- 시크릿 또는 인증정보 입력
- 운영 DB 원본을 테스트/학습 목적으로 사용
- 개인정보 전체를 비마스킹 출력
- 보안 검토 없이 내부망/취약점/소스코드 정보를 외부 전송

## 사고 후보

다음 상황은 보안 또는 개인정보 사고 후보로 분류한다.

- 개인정보가 외부 LLM에 잘못 입력됨
- 프롬프트/응답 로그에 개인정보가 저장됨
- LLM이 개인정보를 과다 출력함
- API Key, 토큰, 비밀번호가 LLM에 입력됨
- 운영 DB 원본이 개발/시험 환경 또는 외부 LLM에 전송됨
- 벡터 DB에 개인정보 원문이 색인됨

## 에스컬레이션 응답

```md
판정: 보류 또는 금지

사유:
- ...

즉시 조치:
- 민감 데이터 제거
- 외부 전송 중단
- 로그 저장 여부 확인
- 시크릿 교체
- 보안/개인정보 담당자 보고

필요 증적:
- 발생 시각
- 입력 데이터 범위
- 사용한 LLM 서비스
- 저장/전송 여부
- 삭제/회수 조치
- 재발방지 대책
```

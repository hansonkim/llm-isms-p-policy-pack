# 금지 데이터 입력 지침

## 원칙

다음 데이터는 승인 없는 외부 LLM, 공개 채팅형 LLM, 불명확한 SaaS AI, 검증되지 않은 API에 입력하지 않는다.

## 금지 데이터

- 주민등록번호
- 여권번호
- 운전면허번호
- 외국인등록번호
- 계좌번호
- 카드번호
- 비밀번호
- 일회용 비밀번호
- 인증토큰
- 세션 쿠키
- API Key
- Secret Key
- SSH Private Key
- DB 접속정보
- 운영 DB 원본
- 개인정보가 포함된 상담내역 원문
- 고객 주문/결제/배송 내역 원문
- 민감정보
- 내부망 IP, 네트워크 구성도, 보안장비 정책
- 취약점 점검 결과 원문
- 비공개 소스코드 중 시크릿 또는 내부 보안정보가 포함된 코드

## LLM 응답 규칙

금지 데이터가 감지되면 다음과 같이 응답한다.

```md
판정: 금지

사유:
요청 또는 입력값에 승인되지 않은 LLM 환경에 입력해서는 안 되는 정보가 포함되어 있습니다.

필요 조치:
- 해당 정보를 제거하거나 마스킹하십시오.
- 이미 외부 LLM에 입력했다면 보안/개인정보 사고 후보로 분류하여 보고하십시오.
- 필요한 경우 키/토큰/비밀번호를 즉시 폐기하고 재발급하십시오.
```

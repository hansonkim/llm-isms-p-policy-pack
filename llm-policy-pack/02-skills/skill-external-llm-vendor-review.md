# Skill: 외부 LLM 및 클라우드 서비스 검토

## Trigger

- OpenAI, Claude, Gemini, Copilot 등 외부 LLM
- SaaS AI
- 클라우드 LLM
- 국외 리전
- 벤더 제공 AI 기능
- 외부 모델 학습/파인튜닝
- 외부 API로 상담/로그/문서 전송

## 확인 항목

1. 외부 서비스 제공자명
2. 서비스 유형: SaaS / PaaS / IaaS / API
3. 입력 데이터 종류
4. 개인정보 포함 여부
5. 프롬프트/응답 저장 여부
6. 모델 학습 사용 여부
7. 데이터 보관기간
8. 데이터 처리 위치와 리전
9. 위탁/재위탁 여부
10. 국외이전 여부
11. 계약서, SLA, DPA, 보안 부속합의 존재 여부
12. 접근권한, 관리자 권한, 로그, 암호화, 삭제 요청 가능 여부
13. 고객 데이터가 벤더의 모델 개선에 사용되는지 여부
14. 장애/사고 통지 절차
15. 계약 종료 시 데이터 삭제 절차

## 판정

- 계약·보안검토 없음: 보류
- 개인정보 원문 입력: 금지 또는 별도 승인 필요
- 국외 리전/국외 사업자 처리: 국외이전 검토 필요
- 모델 학습 사용 가능성 있음: 금지 또는 옵트아웃/계약 명시 필요
- 프롬프트/응답 보관기간 불명확: 보류

## Output

```md
- 서비스명:
- 사용 목적:
- 입력 데이터:
- 개인정보 포함 여부:
- 저장/학습 여부:
- 처리 위치:
- 위탁/재위탁:
- 국외이전:
- 계약/보안문서:
- 위험:
- 필요한 조치:
- 승인 여부:
```

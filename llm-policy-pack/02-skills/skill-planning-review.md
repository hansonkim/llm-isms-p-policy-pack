# Skill: 기획 단계 LLM 검토

## Trigger

- 신규 기능 기획
- AI/LLM 기능 기획
- 고객 데이터 분석
- 마케팅 자동화
- 추천/분류/요약 기능
- 개인정보 입력 화면 설계
- 관리자 화면 설계
- RAG 기반 검색 기능 기획

## 필수 확인 항목

1. 이 기능이 개인정보를 수집하는가?
2. 수집 항목은 목적에 필요한 최소 항목인가?
3. 민감정보 또는 고유식별정보가 포함되는가?
4. 간접수집 또는 제3자 제공 데이터인가?
5. LLM 사업자 또는 외부 API로 전송되는가?
6. 위탁, 재위탁, 국외이전 가능성이 있는가?
7. 개인정보 처리방침 변경이 필요한가?
8. 보유기간과 파기 기준이 정의되었는가?
9. 정보주체 권리 행사 경로가 필요한가?
10. 관리자 화면에서 마스킹, 다운로드 제한, 검색 제한이 필요한가?
11. 프롬프트/응답 로그가 저장되는가?
12. RAG 색인 또는 벡터 DB에 개인정보가 저장되는가?

## 산출 형식

```md
- 개인정보 처리 여부:
- 수집/이용 목적:
- 데이터 항목:
- 정보주체:
- 수집 경로:
- LLM 사용 위치:
- 외부 제공/위탁/국외이전 여부:
- 보유기간:
- 파기 기준:
- 필요한 화면 고지:
- 필요한 승인:
- ISMS-P 관련 기준:
```

## 금지 제안

- 동의나 고지 없이 새로운 목적으로 개인정보 사용
- 개인정보 원문을 외부 LLM으로 자동 전송
- 관리자 화면에서 과도한 고객 검색 허용
- 다운로드/엑셀 Export에 불필요한 개인정보 포함

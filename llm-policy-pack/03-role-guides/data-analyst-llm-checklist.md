# 데이터 분석가 LLM 사용 체크리스트

## 분석 전

- [ ] 분석 목적이 명확한가?
- [ ] 개인정보 원문이 필요한가?
- [ ] 집계값으로 대체 가능한가?
- [ ] 데이터가 마스킹/가명처리 되었는가?
- [ ] 외부 LLM에 전송되는가?
- [ ] 분석 결과가 개인을 식별할 수 있는가?
- [ ] 결과물 보관기간이 정해졌는가?
- [ ] 분석 후 파기 기준이 있는가?

## 권장

- 개인 단위 데이터보다 집계 데이터 사용
- 샘플링 및 익명화
- 식별자 제거
- 외부 전송 전 보안 검토

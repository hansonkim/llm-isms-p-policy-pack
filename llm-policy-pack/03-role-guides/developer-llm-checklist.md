# 개발자 LLM 사용 체크리스트

## 설계 전

- [ ] API가 개인정보를 처리하는가?
- [ ] DB 컬럼에 개인정보/민감정보/고유식별정보가 있는가?
- [ ] 접근권한이 최소권한으로 분리되는가?
- [ ] 조회/변경/삭제/다운로드 권한이 분리되는가?
- [ ] 개인정보 화면 출력은 마스킹되는가?
- [ ] 로그에 개인정보 또는 시크릿이 남는가?
- [ ] 외부 LLM API를 호출하는가?
- [ ] RAG/벡터 DB에 개인정보가 색인되는가?
- [ ] 개발·시험·운영 환경이 분리되는가?
- [ ] 테스트 데이터가 합성 또는 가공 데이터인가?

## 코드 생성 요청 시 금지

- 운영 DB 접속정보 포함
- API Key 포함
- 실제 고객 데이터 포함
- 취약점 점검 결과 원문 포함
- 내부망 구성도 포함

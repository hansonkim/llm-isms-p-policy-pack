# 기획자 LLM 사용 체크리스트

## PRD 작성 전

- [ ] 개인정보를 수집하는 기능인가?
- [ ] 수집 항목은 최소한인가?
- [ ] 민감정보 또는 고유식별정보가 포함되는가?
- [ ] 외부 LLM 또는 외부 API로 전송되는가?
- [ ] 개인정보 처리방침 변경이 필요한가?
- [ ] 위탁, 제3자 제공, 국외이전 가능성이 있는가?
- [ ] 보관기간과 파기 기준이 있는가?
- [ ] 정보주체 권리 행사 경로가 필요한가?
- [ ] 관리자 화면 마스킹 기준이 필요한가?
- [ ] 다운로드/엑셀 Export가 필요한가?

## LLM 요청 시 금지

- 고객정보 원문을 붙여넣기
- 상담내역 원문을 외부 LLM에 입력
- 운영 DB 결과를 그대로 분석 요청
- 개인정보 처리 목적을 정의하지 않고 마케팅 활용 요청

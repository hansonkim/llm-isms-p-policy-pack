# 보안/개인정보 검토자 가이드

## 검토 대상

- 신규 LLM 기능
- 외부 LLM API 도입
- 고객 데이터 분석
- RAG/임베딩/벡터 DB 구축
- 프롬프트/응답 로그 저장
- 운영데이터 테스트 사용
- 관리자 다운로드 기능

## 검토 항목

1. 데이터 분류
2. 개인정보 흐름
3. 위탁/재위탁/국외이전
4. 접근권한
5. 암호화
6. 마스킹
7. 로그/접속기록
8. 보관기간
9. 파기 기준
10. 사고 대응
11. 증적 확보

## 승인 기준

- 처리 목적 명확
- 최소 데이터 원칙 준수
- 외부 전송 검토 완료
- 보관 및 파기 기준 정의
- 책임자와 담당자 명확
- 증적 보관 가능

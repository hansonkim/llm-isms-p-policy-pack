## LLM 적용 관점

| ISMS-P 관점 | LLM 적용 |
|---|---|
| 정보자산 식별 | LLM 서비스, 프롬프트 로그, 벡터 DB, API Key 식별 |
| 흐름분석 | 데이터가 LLM으로 들어가고 저장/출력되는 흐름 문서화 |
| 외부자 보안 | 외부 LLM 벤더 계약, 위탁, 재위탁, 보안점검 |
| 개발 보안 | LLM 기능의 보안 요구사항, 취약점 검토 |
| 시험 데이터 보안 | 운영데이터의 LLM 테스트 사용 제한 |
| 로그관리 | 프롬프트/응답 로그 보관, 접근통제, 점검 |
| 클라우드 보안 | LLM SaaS/API의 리전, 권한, 설정, 로그 |
| 파기 | 프롬프트/응답/첨부파일/벡터 DB 파기 |

# 마스킹 표준

| 항목 | 표준 |
|---|---|
| 성명 | 가운데 글자 마스킹. 예: 김*주 |
| 휴대전화 | 국번 마스킹. 예: 010-****-1234 |
| 이메일 | ID 일부 마스킹. 예: ab****@example.com |
| 주소 | 상세주소 마스킹. 예: 서울시 강남구 테헤란로 *** |
| 주민등록번호 | 뒤 7자리 마스킹. 예: 900101-******* |
| 카드번호 | 중간 번호 마스킹. 예: 1234-56**-****-1234 |
| 계좌번호 | 중간 번호 마스킹. 예: 110-***-****** |
| IP 주소 | 일부 대역 마스킹. 예: 123.123.***.123 |

## 적용 원칙

1. 업무 목적상 원문이 필요하지 않으면 마스킹한다.
2. 관리자 화면, 다운로드, 보고서, 로그, LLM 출력에 동일 기준을 적용한다.
3. 숨겨진 필드, HTML 소스, 원본 JSON에 원문이 남지 않도록 확인한다.
4. 주민등록번호, 비밀번호, 인증토큰, API Key는 출력하지 않는다.

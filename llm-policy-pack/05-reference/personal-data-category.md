# 개인정보 및 중요정보 분류

## 개인정보

- 이름
- 생년월일
- 전화번호
- 이메일
- 주소
- 계정 ID
- 고객번호
- 주문내역
- 상담내역
- 접속기록
- 위치정보

## 민감정보

- 건강정보
- 유전정보
- 범죄경력
- 정치적 견해
- 종교
- 노동조합 가입 여부
- 생체인식정보 중 민감한 처리가 가능한 정보

## 고유식별정보

- 주민등록번호
- 여권번호
- 운전면허번호
- 외국인등록번호

## 인증정보/시크릿

- 비밀번호
- OTP
- 세션 쿠키
- Access Token
- Refresh Token
- API Key
- Secret Key
- SSH Private Key
- DB 접속정보

## 중요정보

- 내부망 구성도
- 보안장비 정책
- 취약점 점검 결과
- 영업비밀
- 재무정보
- 인사정보
- 비공개 소스코드
